# Math API Assignment

This is a simple REST API built using Python's built-in HTTP server.  
It performs basic math operations: addition, subtraction, and multiplication.

## Features
- Addition endpoint
- Subtraction endpoint
- Multiplication endpoint
- Returns results in JSON format

## API Endpoints
- `/add?a=5&b=3`
- `/subtract?a=5&b=3`
- `/multiply?a=5&b=3`

## How to Run

```bash
python3 server.py
